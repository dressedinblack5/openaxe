## CONVENTIONS

- **Generated code is read-only.** Edit spec or build script, never hand-edit `gen/` files.
- **Spec source:** `bun dev generate` from the `openaxe` package outputs the OpenAPI JSON. Build falls back to committed `openapi.json` if generation fails (CI guard).
- **SSE type patch:** `build.ts` strips `TError` from `ServerSentEventsResult` generic вЂ” `@hey-api/openapi-ts` bug workaround.
- **Export transform:** `publish.ts` rewrites `./src/` paths to `./dist/` in `exports` before publishing.
- **Idempotent publish:** script checks `npm view` before publishing; skips if version exists.
- **Entrypoints:** `"."`, `"./client"`, `"./server"`, `"./v2"`, `"./v2/client"`, `"./v2/server"`, `"./v2/types"`.
